Play-module-less

Less module for the Play framework

Download this project as a .zip file Download this project as a tar.gz file

Play LESS module

This module allows you to use LESS (http://www.lesscss.org) directly in your Play! 1.2.5+ project. The documentation is in /documentation.

Caveat emptor

The only reason I did this release was to provide myself with updated versions of the play less module. It makes use of unofficial forks Asual's lesscss-engine. It does not come with any support nor any promise that it will have the same functionality as Asual's future releases.
